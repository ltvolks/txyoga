txYoga

txYoga is an alpha-quality REST toolkit for Twisted.

Quickstart

  1. pip install git://github.com/lvh/txyoga.git

  2. Read the tutorial.

  3. Tell me what's wrong with it.

Documentation

The documentation is graciously hosted by ReadTheDocs. They also provide an easy URL for them at txyoga.rtfd.org.

Development

Development on txYoga happens primarily through Github. Fork me and send me a pull request!

If you find any bugsundocumented features of txYoga, please file a ticket on the issue tracker so it can be fixed.
